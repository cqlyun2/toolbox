# 雲歌工具小站 - 平台规划文档

## 一、平台概述

**目标**：打造一站式 AI 工具服务平台
**域名**：yunge.org.cn

---

## 二、五大板块

### 2.1 板块一：Web小工具 (www.yunge.org.cn)

**定位**：在线工具导航站，类似"帮小忙"
**现状**：已完成基础开发
- 技术栈：Vue 3 + Vite + TailwindCSS + Express
- 功能：图片处理、PDF工具、文本工具、AI工具
- 会员：手机号登录 + VIP 会员系统
- AI：硅基流动 API (Qwen2.5-7B)

**待完成**：
- [ ] 完善会员系统（支付接口）
- [ ] 优化 AI 生成效果

## 五、注意事项

1. **子域名独立部署**：每个板块独立项目，互不干扰
2. **统一登录**：考虑 SSO 单点登录
3. **SEO 优化**：静态页面注意 SEO
4. **移动端适配**：响应式设计
