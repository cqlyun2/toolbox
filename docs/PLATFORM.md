# 雲歌工具小站 - 平台规划文档

## 一、平台概述

**目标**：打造一站式 AI 工具服务平台
**域名**：yunge.org.cn

---

## 二、四大板块

### 2.1 板块一：在线工具 (www.yunge.org.cn)

**现状**：已完成基础开发
- 技术栈：Vue 3 + Vite + TailwindCSS + Express
- 功能：图片处理、PDF工具、文本工具、AI工具、渝工小工具
- 会员：手机号登录 + VIP 会员系统
- AI：硅基流动 API (Qwen2.5-7B)

**待完成**：
- [ ] 完善会员系统（支付接口）
- [ ] 扩展渝工小工具（剩余20+工具）
- [ ] 优化 AI 生成效果

---

### 2.2 板块二：Skills 商店 (skills.yunge.org.cn)

**定位**：AI 助手 Skills 分享平台

**内容**：
- 钉钉 Skills
- 飞书 Skills
- 企业微信 Skills

**功能**：
- Skill 展示、搜索
- 在线预览
- 一键复制/下载
- 作者主页
- 评价/评分

**技术**：
- Vue 3 / Next.js
- 用户系统（可复用会员）
- 支付（赞赏/付费下载）

---

### 2.3 板块三：工作流 (workflows.yunge.org.cn)

**定位**：AI 工作流模板下载

**内容**：
- ChatGPT 工作流
- Claude 工作流
- Midjourney 工作流
- ComfyUI 工作流
- 其他 AI 工作流

**功能**：
- 模板展示、分类
- 在线预览
- 下载（JSON/配置文件）
- 工作流教程

**技术**：
- 静态站点 / Next.js
- 文件存储

---

### 2.4 板块四：Agent (agent.yunge.org.cn)

**定位**：AI Agent/助手下载

**内容**：
- GPTs 助手
- Claude Agents
- 自定义 AI 助手

**功能**：
- Agent 展示、搜索
- 配置说明
- 下载配置
- 使用教程

**技术**：
- 静态站点 / Next.js

---

## 五、注意事项

1. **子域名独立部署**：每个板块独立项目，互不干扰
2. **统一登录**：考虑 SSO 单点登录
3. **SEO 优化**：静态页面注意 SEO
4. **移动端适配**：响应式设计
